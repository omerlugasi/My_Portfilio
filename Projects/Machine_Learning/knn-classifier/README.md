# KNN Classifier

A Machine Learning project implementing a K-Nearest Neighbors classifier in Python.

## Overview

This project implements a KNN classifier from scratch using NumPy.

The classifier supports:
- Minkowski distance
- configurable `k`
- configurable `p` distance parameter
- vectorized distance calculation
- majority voting
- tie-breaking logic
- command-line execution

## Technologies

- Python
- NumPy
- Pandas
- Machine Learning
