# Perceptron Classifier

A Machine Learning project implementing a multiclass perceptron classifier in Python.

## Overview

This project implements a multiclass perceptron algorithm from scratch
using NumPy.

The model performs iterative weight updates until convergence
on linearly separable datasets.

The implementation includes:
- multiclass classification
- perceptron training
- weight updates
- prediction pipeline
- NumPy-based computations
