# Statistical Inference on Health Data

Statistical inference analysis project based on the BRFSS 2021 health survey dataset.

The project focuses on estimating population parameters and performing hypothesis testing using classical statistical inference methods.

Main topics:
- Confidence intervals
- Hypothesis testing
- Pooled t-tests
- Wald tests
- Likelihood Ratio Tests (LRT)
- Sampling behavior analysis
- Confidence interval convergence
- Statistical simulations

The analysis compares BMI distributions between individuals with and without a history of heart attack using real-world health survey data.

## Technologies Used

- Python
- Pandas
- NumPy
- SciPy
- Matplotlib
- Seaborn
