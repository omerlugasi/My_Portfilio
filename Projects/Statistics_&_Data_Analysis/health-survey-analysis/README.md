# Health Survey Analysis

Exploratory data analysis and preprocessing project based on the BRFSS 2021 health survey dataset published by the CDC.

The project focuses on:
- Data cleaning and preprocessing
- Missing value handling
- Feature selection
- Statistical visualization
- Exploratory Data Analysis (EDA)
- Formulating statistical research questions

The dataset contains demographic, behavioral, and health-related variables such as:
- Age
- BMI
- Smoking status
- Income category
- Mental health indicators
- Chronic disease history

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- PyReadStat
