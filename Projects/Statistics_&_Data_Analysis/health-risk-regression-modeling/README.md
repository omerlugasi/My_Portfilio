# Health Risk Regression Modeling

Statistical modeling project focused on analyzing relationships between health-related risk factors and mental health outcomes using linear and logistic regression techniques.

The project examines how variables such as BMI, smoking status, and high cholesterol are associated with:
- Poor mental health days
- History of depression

Main topics:
- Linear regression
- Logistic regression
- ANOVA
- Correlation analysis
- Residual diagnostics
- Model assumptions
- Confidence intervals
- Prediction intervals
- Interaction effects
- Model evaluation and interpretation

The analysis is based on the BRFSS 2021 health survey dataset published by the CDC.

## Technologies Used

- Python
- Pandas
- NumPy
- SciPy
- Statsmodels
- Matplotlib
- Seaborn
