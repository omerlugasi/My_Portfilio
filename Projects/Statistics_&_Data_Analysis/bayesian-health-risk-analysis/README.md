# Bayesian Health Risk Analysis

Bayesian statistics project focused on health-risk estimation, posterior inference, and missing-data analysis using real-world health survey data.

Main topics:
- Bayesian estimation
- Posterior distributions
- Uniform and Jeffreys priors
- Historical priors
- Credible intervals
- Bootstrap confidence intervals
- Monte Carlo simulation
- Missing value imputation
- Regression analysis with incomplete data

The analysis is based on the BRFSS 2021 health survey dataset published by the CDC.

## Technologies Used

- Python
- Pandas
- NumPy
- SciPy
- Statsmodels
- Matplotlib
