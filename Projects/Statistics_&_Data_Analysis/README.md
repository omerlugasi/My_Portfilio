# Statistics and Data Analysis Projects

This repository contains a collection of applied statistics and data analysis projects developed as part of the *Statistics 2* course.

The projects focus on real-world health survey data analysis using statistical inference, regression modeling, bootstrap methods, and Bayesian analysis techniques.

All projects are based on the publicly available **BRFSS 2021** dataset published by the CDC (Centers for Disease Control and Prevention).

---

# Repository Structure

## 1. health-survey-analysis
Exploratory data analysis and preprocessing pipeline built on top of the BRFSS 2021 health survey dataset.

Main topics:
- Data cleaning and preprocessing
- Missing value handling
- Feature selection
- Exploratory Data Analysis (EDA)
- Statistical visualizations
- Research question formulation

---

## 2. statistical-inference-on-health-data
Classical statistical inference methods applied to health-related variables.

Main topics:
- Confidence intervals
- Hypothesis testing
- F-tests
- Sampling distributions
- Group comparison analysis

---

## 3. mental-health-regression-analysis
Regression-based analysis of relationships between demographic and health-related variables.

Main topics:
- Linear regression
- Logistic regression
- Correlation analysis
- Model interpretation
- Health risk factor analysis

---

## 4. bootstrap-confidence-interval-analysis
Implementation and comparison of bootstrap-based confidence interval estimation methods.

Main topics:
- Bootstrap sampling
- Percentile intervals
- Pivot intervals
- Covariance matrix approximation
- Statistical interval comparison

---

## 5. bayesian-health-risk-analysis
Bayesian inference methods applied to health-risk estimation problems.

Main topics:
- Bayesian estimation
- Prior and posterior distributions
- Jeffreys prior
- Credible intervals
- Bayesian vs. frequentist comparison

---

# Dataset

The projects use a cleaned subset of the **Behavioral Risk Factor Surveillance System (BRFSS) 2021** dataset.

The BRFSS is a large-scale public health survey conducted annually by the CDC across the United States and includes demographic, behavioral, and medical variables.

Dataset source:
https://www.cdc.gov/brfss/annual_data/annual_2021.html
