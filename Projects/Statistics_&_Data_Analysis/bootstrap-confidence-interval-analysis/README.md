# Bootstrap Confidence Interval Analysis

Statistical estimation project focused on bootstrap-based confidence interval methods and statistical interval comparison using real-world health survey data.

Main topics:
- Bootstrap sampling
- Percentile intervals
- Pivot intervals
- Covariance matrix approximation
- Prediction intervals
- Wald tests
- Permutation testing
- Statistical interval comparison

The analysis is based on the BRFSS 2021 health survey dataset published by the CDC.

## Technologies Used

- Python
- Pandas
- NumPy
- SciPy
- Statsmodels
- Matplotlib
